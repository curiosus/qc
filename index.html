Basic Operations/Gates

Not

Initialze one qbit to zero and print. Wait 3 seconds and appl the not gate and print. 0 to 1. Wait 3 seconds more and apply the not gate again and print. 1 to 0 

applyNot = () => {
    qc.not();
    const result = qc.read();
    qc.print(result);
    setTimeout(applyNot, 3000);
};

//initialize one qbit
qc.reset(1);

//assign it value of 0
qc.write(0);

//read the qbit and store result
const result = qc.read();

//print the result
qc.print(result);

//apply the not gate recursivley
setTimeout(applyNot, 3000);

Hadamard (had) gate

Hadamard creates a uniform superposition where each outcome |0> or |1> is equally likely. Here we are applying had to |0> to create a superposition. Next we apply had to the qbit in superpositon, which results in |0> 

let hasRun = false;
applyHad = () => {
    qc.had();
    if (!hasRun) {
        hasRun = true;
        //hadamard applied to existing superposition
        setTimeout(applyHad, 3000);
    }
};

//hadamard applied to initial state of |0>
qc.reset(1);
qc.write(0);
const result = qc.read();
setTimeout(applyHad, 3000);
